Build the property listings system with modern UI and image upload.

Requirements:

* Create listing form
* Edit and delete listings
* Multiple image upload using Supabase Storage
* Image preview before upload
* Drag and drop image upload
* Validation and error handling

Fields:

* Title
* Description
* Price
* Property type (room, apartment)
* Area (Dubai locations)
* Bedrooms
* Bathrooms
* Furnished

UI:

* Airbnb-style property cards
* Responsive grid
* Mobile-first design
* Skeleton loading
* Empty states
* Clean modern Tailwind UI

Security:

* Only listing owner can edit or delete
* Supabase RLS enforcement

Architecture:

* Server actions
* Supabase database
* Supabase storage
* Reusable components

Output:

* Listing form
* Listing cards
* Listing dashboard page
* Image upload system
* Storage bucket setup instructions
